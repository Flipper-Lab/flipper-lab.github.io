Участие в разработке

Как помочь проекту?

Помоги перевести оригинальную документацию на русский язык

  • Исправляй ошибки в существующих переводах
  • Переводи новые разделы
  • Улучшай формулировки

Добавляй полезные материалы и примеры

  • Пиши гайды и туториалы
  • Создавай схемы и диаграммы
  • Добавляй скриншоты и примеры кода

Быстрый старт для контрибьютора

1. Создайте форк репозитория

Нажмите "Fork" в правом верхнем углу на GitHub

2. Клонируйте репозиторий
git clone https://github.com/ВАШ_АККАУНТ/flipper-lab.github.io.git
3. Создайте ветку для изменений
git checkout -b feature/название-раздела
4. Внесите изменения

Редактируйте файлы в папке content/

5. Запушьте изменения
git add .
git commit -m "Добавлен раздел: Название"
git push origin feature/название-раздела
6. Создайте Pull Request

В интерфейсе GitHub создайте Pull Request из вашей ветки в main

Рекомендации по переводу

Что делать:
  • Сохраняйте техническую точность - термины переводите корректно
  • Используйте простой язык - избегайте сложных конструкций
  • Добавляйте контекст - если термин непонятен, дайте пояснение
  • Сохраняйте структуру - заголовки, списки, картинки должны быть на месте
Чего избегать:
  • Буквальный перевод - переводите смысл, а не слова
  • Слишком сложные термины - если есть простой аналог, используйте его
  • Изменение структуры - не меняйте расположение разделов без необходимости
