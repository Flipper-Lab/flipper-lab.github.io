Запись данных на T5577 карты

В общем случае, 125 kHz RFID карты и брелоки предназначены только для чтения. Если они перезаписываемые, они обычно имеют пароль и используют чип, несовместимый с Flipper Zero для записи. Однако существуют перезаписываемые чистые карты T5577, которые могут быть запрограммированы для эмуляции карт с различными низкочастотными RFID протоколами.

Вы можете записать сохранённые и добавленные вручную 125 kHz карты на чистые карты T5577. Эти чистые карты представлены в различных формах и формах, таких как карты, брелоки, наклейки и микрочипы для животных. Flipper Zero способен записывать данные со всеми поддерживаемыми низкочастотными RFID протоколами.

На этой странице вы узнаете как записывать данные на чистые карты T5577 и что делать, если процедура не удалась.

Как записывать данные на чистые карты T5577

Чтобы записать данные на карту, выполните следующие действия:

  • Перейдите в Main Menu -> 125 kHz RFID -> Saved.
  • Выберите карту которую вы хотите записать, затем нажмите Write.
  • Поднесите ваш Flipper Zero к чистой карте T5577, задняя часть устройства должна быть обращена к карте.
  • После записи данных на чистую карту, устройство отобразит сообщение "Successfully written".

Ваш Flipper Zero может отобразить сообщение "Successfully written" при записи на карту только для чтения. Это может произойти если идентичная информация уже находится на карте только для чтения.

Если запись не удалась

  • Чистая карта T5577 может быть защищена паролем.
  • Вы пытаетесь записать на RFID карту только для чтения, а не на чистую карту T5577.
