Чтение 125 kHz RFID карт

Flipper Zero позволяет вам читать, сохранять и эмулировать 125 kHz RFID карты. 125 kHz RFID карта - это транспондер, который хранит уникальный идентификационный номер. При сканировании считывателем, 125 kHz карта передаёт свой ID номер. Если карта использует поддерживаемый протокол, Flipper Zero может прочитать и сохранить ID номер.

На этой странице вы узнаете как читать, сохранять и эмулировать 125 kHz RFID карты с вашим Flipper Zero.

Как читать 125 kHz карты

Чтобы прочитать и сохранить данные 125 kHz карты, выполните следующие действия:

  • Перейдите в Main Menu -> 125 kHz RFID.
    • Нажмите Read, затем поднесите карту к задней части вашего Flipper Zero.
    • Не двигайте карту во время чтения. Процесс чтения может занять несколько секунд потому что Flipper Zero переключает кодировки и пытается подобрать протокол карты из списка поддерживаемых протоколов.
    • Во время чтения Flipper Zero переключается между ASK и PSK кодировками каждые три секунды для чтения данных с 125 kHz RFID карты.
  • После завершения чтения вы увидите данные карты на экране.
  • Чтобы сохранить карту, перейдите в More -> Save.
  • Назовите карту, затем нажмите Save.

Если чтение не удалось

  • Карта может использовать NFC технологию. -> Прочитайте карту с помощью NFC приложения.
  • Flipper Zero переключает кодировки каждые три секунды. Некоторые карты могут не быть прочитаны, так как это может потребовать до 10 секунд для чтения данных. -> Прочитайте карту с предварительно выбранной ASK или PSK кодировкой.

Чтение с предварительно выбранной кодировкой

Flipper Zero позволяет вам читать 125 kHz RFID карты с предварительно выбранной ASK или PSK кодировкой.

Чтобы прочитать и сохранить данные 125 kHz карты с предварительно выбранной кодировкой, выполните следующие действия:

  • Перейдите в Main Menu -> 125 kHz RFID -> Extra Actions.
  • Выберите Read ASK или Read PSK.
  • Поднесите карту к задней части вашего Flipper Zero.
  • После чтения, перейдите в More -> Save.
  • Назовите карту, затем нажмите Save.

Эмуляция 125 kHz RFID карт

Flipper Zero может эмулировать сохранённые 125 kHz RFID карты, выполнив следующие действия:

  • Перейдите в Main Menu -> 125 kHz RFID -> Saved.
  • Выберите карту которую вы хотите эмулировать, затем нажмите Emulate.
  • Поднесите ваш Flipper Zero к считывателю, задняя часть устройства должна быть обращена к считывателю.
