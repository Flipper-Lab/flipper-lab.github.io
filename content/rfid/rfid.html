125 kHz RFID

Flipper Zero поддерживает технологию низкочастотной (LF) радиочастотной идентификации (RFID), которая применяется в системах контроля доступа, чипах для животных и системах отслеживания цепочек поставок. В отличие от NFC карт, LF RFID карты обычно не обеспечивают высокий уровень безопасности. Эта технология представлена во многих форм-факторах, таких как пластиковые карты, брелоки, метки, браслеты и микрочипы для животных. Flipper Zero имеет модуль низкочастотной RFID, который может читать, сохранять, эмулировать и записывать LF RFID карты.

Вставьте microSD карту чтобы использовать приложение 125 kHz RFID

Меню 125 kHz RFID

Вы можете получить доступ к приложению 125 kHz RFID через Main Menu. В приложении вы можете читать, сохранять, эмулировать, записывать и генерировать новые LF RFID карты.

  • Read - читает и сохраняет LF RFID карты.
  • Saved - список сохранённых карт, которые можно эмулировать и записать на перезаписываемую карту.
  • Add Manually - генерирует новые виртуальные LF RFID карты путём ввода ID карты.
  • Extra Actions - позволяет читать LF RFID карты с предварительно выбранной ASK или PSK кодировкой.

Аппаратное обеспечение 125 kHz RFID

Flipper Zero имеет встроенную поддержку RFID с низкочастотной антенной, расположенной на задней части Flipper Zero. Микроконтроллер STM32WB55 используется для функциональности 125 kHz RFID.

Низкочастотная антенна 125 kHz размещена на двухдиапазонной RFID антенне рядом с высокочастотной антенной 13.56 МГц.
