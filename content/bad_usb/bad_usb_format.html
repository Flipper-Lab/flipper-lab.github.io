Официальная документация

Синтаксис команды

Приложение BadUsb использует расширенный синтаксис Duckyscript. Оно совместимо с классическими скриптами USB Rubber Ducky 1.0, но предоставляет ряд дополнительных команд и функций, таких как настраиваемый USB-идентификатор, метод ввода ALT+Numpad, команда SYSRQ и больше функциональных клавиш.

Формат файла скрипта

Приложение BadUsb может выполнять только текстовые скрипты из TXT-файлов, компиляция не требуется. Поддерживаются символы \n и \r\n в конце строк. Пустые строки разрешены. Для отступов строк можно использовать пробелы или табуляции.
